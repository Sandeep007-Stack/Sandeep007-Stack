## 🧭 About Me

I'm a full-stack / frontend developer working a full-time job by day and shipping side projects by night. I build **product-first**, validate before I over-engineer, and lean hard into React/Next.js on the frontend and Node.js/Python on the backend. Lately I've been deep in ML/AI infra — running PyTorch natively on Apple Silicon, and using the Claude API to power both my apps and my content pipeline.

- 🔭 Currently building **CostShift.io**, **FarmMind AI**, and **APICraft**
- ✍️ Writing and automating content for **[stackdevlife.com](https://stackdevlife.com)** — 500+ newsletter subscribers
- 🎥 Making dev content on **Stack Dev Life** (YouTube)
- 🧠 Learning ML/AI seriously — Fast.ai, Coursera MLOps, building from-scratch LLM understanding
- 🛠️ Big on stress-testing ideas before building — Stack Exchange Code Review regular
- 📍 Based in Central India
